Project 4.0

Project 4.0 was(op de stage na) het grootste project in de opleiding. Het project werd uitgevoerd met 5 andere studenten. Bij dit project werd er van ons verwacht dat we eerst gedurende 3 maanden, een analyse doen van de opdracht en scherm ontwerpen maken. Hierbij ging het echt om het leren werken rond de agile manier

Na de analyse, kwamen er 3 weken van programmeren. Ik heb vooral gewerkt aan het back-end gedeelte van het project. Dus werken aan security, unit testen en api calls. Het meeste tijd werd gestoken in het uitzoeken van unit testen met jwt tokens.

Als de back-end grotendeels klaar was ben ik ook mee komen helpen in de front-end. Daar ben ik bezig geweest met form validatie en zoeken naar bugs.
